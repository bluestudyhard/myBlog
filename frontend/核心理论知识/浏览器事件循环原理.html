                        # 线程与进程

                        # 什么是进程

                        进程简单来说可以理解为是一片内存空间。
                        比如说我手机开了一个微信，一个 QQ，那么系统就会为这两块应用开辟内存空间。
                        一个应用会有很多的进程，比如说浏览器的，渲染进程，浏览器进程，网络进程，GPU 进程等。

                        # 什么是线程

                        线程，简单来理解就是跑代码的，在进程中负责执行相关的操作的，就是线程，一个进程最少有一个线程。
                        比如说在渲染进程里，会有一个渲染主线程负责渲染，
                        网络进程中，网络线程干活
